## ðŸŽ¬ Recording Tips

### Before Recording

1. **Prepare Test Data**
   - Create sample products
   - Generate test orders
   - Set up mock disputes
   - Use realistic names and data

2. **Browser Setup**
   - Clear cache and cookies
   - Close unnecessary tabs
   - Full screen mode (F11)
   - Hide bookmarks bar
   - Zoom: 100%

3. **Screen Resolution**
   - 1920x1080 (Full HD)
   - or 1280x720 (HD)
   - 16:9 aspect ratio

4. **Audio Setup**
   - Test microphone quality
   - Quiet environment
   - Use pop filter
   - Record separate audio if needed

### During Recording

1. **Cursor Highlights**
   - Use cursor highlighting tool
   - Slow, deliberate movements
   - Pause briefly before clicks

2. **Pacing**
   - Speak slowly and clearly
   - Pause between major steps
   - Leave 2-3 seconds silence for transitions

3. **Annotation Tools**
   - Highlight important buttons
   - Use arrows for navigation
   - Add text callouts for tips

### After Recording

1. **Editing Checklist**
   - Remove mistakes and pauses
   - Add background music (soft, non-distracting)
   - Insert transitions between scenes
   - Add subtitles/captions
   - Include timestamps in description

2. **Export Settings**
   - Format: MP4
   - Codec: H.264
   - Resolution: 1080p
   - Frame rate: 30fps
   - Bitrate: 8-12 Mbps

3. **Distribution**
   - Upload to YouTube
   - Embed in Help Center
   - Share on social media
   - Include in onboarding emails
